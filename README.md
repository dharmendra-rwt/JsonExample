# C# JSON and NuGet Assignment

## Project Description
This project demonstrates JSON file creation, JSON deserialization, inheritance, and use of Newtonsoft.Json in C#.

## Tasks Completed

### Task 1
Created manual JSON file (user.json).

### Task 2
Added multiple user entries into the JSON file.

### Task 3
Deserialized JSON data into C# objects and displayed data using loops.

### Task 4
Implemented inheritance using:

- User (base class)
- Admin : User
- RegularUser : User

### Task 5
Created second JSON file (user_types.json) and deserialized its data.
